Transforming the Blue Marble into something you can serve.

1. downloand images

2. create geotiffs by adding geobounds for the imagines

gdal_translate -of GTiff -a_srs EPSG:4326 -a_ullr -180 90 -90 0  world.200406.3x21600x21600.A1.jpg A1.tif
gdal_translate -of GTiff -a_srs EPSG:4326 -a_ullr -180 0 -90 -90  world.200406.3x21600x21600.A2.jpg A2.tif
gdal_translate -of GTiff -a_srs EPSG:4326 -a_ullr -90 90 0 0  world.200406.3x21600x21600.B1.jpg B1.tif
gdal_translate -of GTiff -a_srs EPSG:4326 -a_ullr -90 0 0 -90  world.200406.3x21600x21600.B2.jpg B2.tif
gdal_translate -of GTiff -a_srs EPSG:4326 -a_ullr 0 90 90 0  world.200406.3x21600x21600.C1.jpg C1.tif
gdal_translate -of GTiff -a_srs EPSG:4326 -a_ullr 0 0 90 -90  world.200406.3x21600x21600.C2.jpg C2.tif
gdal_translate -of GTiff -a_srs EPSG:4326 -a_ullr 90 90 180 0  world.200406.3x21600x21600.D1.jpg D1.tif
gdal_translate -of GTiff -a_srs EPSG:4326 -a_ullr 90 0 180 -90  world.200406.3x21600x21600.D2.jpg D2.tif

3. create shape file

gdaltindex -write_absolute_path bluemarble-index.shp A1.tif A2.tif B1.tif B2.tif C1.tif C2.tif D1.tif D2.tif


4. create the mapfile seen in cay

5. maybe run shptree on the shapefile to create an index
